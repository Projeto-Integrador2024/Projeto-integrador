	Este sistema faz o cadastro e consulta de diferentes tipos de pessoas em um sistema de uma grande universidade. Será desenvolvido usando um diagrama de casos de uso representando os cenários de um sistema de gestão escolar contendo:
- Pessoa física
- Pessoa jurídica
- Professores
- Fornecedores
- Alunos

	Esse sistema será utilizado por exemplo por professores e alunos, para fazer matrícula, consultar notas, consultar pendências financeiras, entre outras.

	Este sistema será usado para cadastrar, armazenar e consultar informações de professores, alunos, pessoas físicas, pessoas jurídicas, fornecedores,  também poderá realizar tarefas como por exemplo realizar matrículas de alunos. Os principais usuários deste sistema serão professores, alunos, pessoas físicas, pessoas jurídicas e fornecedores, que ao utilizarem o sistema terão como principais vantagens economia de tempo e aumento de produtividade. 

Objetivos:
	O sistema tem como objetivo facilitar a vida dos usuários (pessoas físicas, pessoas júridicas, professores, alunos, fornecedores) uma vez que utilizando o sistema eles terão economia de tempo, pois no sistema  podem fazer cadastro, fazer consultas diversas, entre outros.
